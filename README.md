# SaaS Acer

SaaS Acer is a SaaS readiness analyzer that evaluates a startup's growth foundation based on factors like target audience, email lists, landing pages, and social presence.

The tool uses a weighted scoring system to generate a score out of 100 and provides feedback based on the results.

## Features

- Interactive SaaS assessment
- Weighted scoring system
- Dynamic results and feedback
- Animated progress bar
- Responsive design

## Built With

- HTML
- CSS
- JavaScript

## Concepts Practiced

- DOM manipulation
- Event listeners
- Conditional logic
- Dynamic UI updates
- Flexbox layouts
